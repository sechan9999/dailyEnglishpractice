# Daily English Interview Practice for Data Scientists

This repository contains a collection of daily English interview practice sessions designed for Data Scientist roles. Each session focuses on a common data science interview question, providing a model answer, key vocabulary, pronunciation tips, and a practice prompt.

## Topics Covered So Far:

*   **Day 1:** Bias-Variance Tradeoff
*   **Day 2:** Supervised vs. Unsupervised Learning
*   **Day 3:** Cross-validation
*   **Day 4:** How a Random Forest works

Stay tuned for more daily practice sessions to help you ace your data science interviews!
